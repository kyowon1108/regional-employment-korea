## 📋 프로젝트 개요

본 프로젝트는 **2021-2023년 완전한 고용률 데이터를 보유한 153개 지자체**를 대상으로 E9 비자 체류자수와 지역별 고용률 간의 관계를 엄밀한 패널분석 방법론으로 분석합니다.

### 🎯 주요 연구 질문
- E9 체류자수 변화가 지역 고용률에 미치는 실질적 효과는?
- 제조업 집중도가 E9-고용률 관계를 조절하는가?
- 데이터 품질 중심 접근법의 분석적 가치는?

### 🔍 핵심 발견사항
- **E9 효과**: 통계적으로 유의하지 않음 (p=0.281)
- **실질적 효과**: E9 1% 증가 → 고용률 0.0007%p 증가 (경제적으로 미미)
- **제조업 집중도**: 유일하게 유의한 변수 (p=0.027)
- **모델 설명력**: Within R² = 0.7% (낮은 설명력)

## 📈 분석 방법론

### 데이터 선별 과정
1. **1차 필터링**: Excel에서 2019-2020년 구조 기준 지자체 식별
2. **2차 검증**: 2021-2023년 완전한 고용률 데이터 보유 확인
3. **최종 선정**: 153개 지자체 (전체의 약 67%)

### 패널분석 모형
```
고용률ᵢₜ = αᵢ + δₜ + β₁(E9_log)ᵢₜ + β₂(manufacturing_conc)ᵢₜ + β₃(employee_density_log)ᵢₜ + εᵢₜ
```

**여기서:**
- `αᵢ` = 지자체 고정효과 (지역 특성 통제)
- `δₜ` = 시간 고정효과 (공통 시간 트렌드 통제)
- Within Transformation 적용으로 순수한 within variation 활용

# 고품질 지자체 대상 E9-고용률 관계 패널분석 보고서

## 📋 분석 개요

본 보고서는 **2021-2023년 고용률 데이터를 완전히 보유한 153개 지자체를 대상**으로 E9 체류자수와 지역별 고용률 간의 관계를 Two-way Fixed Effects 패널분석을 통해 엄밀하게 분석한 연구입니다. 데이터 품질 중심의 선별적 접근법을 통해 보다 신뢰할 수 있는 실증적 근거를 제시하고자 합니다.

---

## 🎯 연구 목적 및 가설

### 주요 연구 질문
1. **E9-고용률 관계**: E9 체류자수 변화가 지역 고용률에 미치는 순효과는?
2. **산업구조 조절효과**: 제조업 집중도가 E9 효과를 조절하는가?
3. **정책적 실효성**: E9 비자 정책이 지역 고용률 개선에 기여하는 정도는?

### 연구 가설
- **H1**: E9 체류자수 증가는 지역 고용률에 양(+)의 영향을 미칠 것이다
- **H2**: 제조업 집중 지역에서 E9 효과가 더 클 것이다
- **H3**: 지역별 내재적 특성을 통제해도 E9 효과는 유의할 것이다

## 📊 데이터 및 분석 대상

### 데이터 선별 과정
1. **1차 필터링**: Excel 데이터에서 2019-2020년 구조 기준 지자체 식별
2. **2차 검증**: 통합 데이터에서 2021-2023년 완전한 고용률 데이터 보유 확인
3. **최종 선정**: 153개 지자체 (전체의 약 67% 수준)

### 최종 분석 데이터
- **분석 지자체**: 153개 (고품질 데이터 보유)
- **분석 기간**: 2021-2023년 (3개년)
- **데이터 구조**: 반기별 패널데이터 (연평균화 미적용)
- **총 관측치**: 924개 (153개 × 3년 × 2반기)
- **데이터 특성**: 완전 균형 패널 (결측치 없음)

### 주요 변수
#### 종속변수
- **고용률**: 지역별 반기별 고용률 (%, 2021-2023년만 유효)

#### 핵심 독립변수
- **E9_log**: E9 체류자수의 자연로그 변환값
- **manufacturing_concentration**: 제조업 집중도 (제조업/전체종사자)
- **employee_density_log**: 종사자 밀도의 자연로그 변환값

## 🔍 분석 방법론

### Two-way Fixed Effects 패널모형
```
고용률ᵢₜ = αᵢ + δₜ + β₁(E9_log)ᵢₜ + β₂(manufacturing_conc)ᵢₜ + β₃(employee_density_log)ᵢₜ + εᵢₜ
```

**여기서:**
- i = 지자체 (1, 2, ..., 153)
- t = 시점 (2021.1반기 ~ 2023.2반기)
- αᵢ = 지자체 고정효과 (지역 특성 통제)
- δₜ = 시간 고정효과 (공통 시간 트렌드 통제)

### Within Transformation 적용
각 변수에서 개체별 평균과 시간별 평균을 제거하여 순수한 within variation만 활용

## 🔍 결과 해석 및 논의

### 1. E9 체류자수 효과 분석

#### 통계적 측면
- **계수 추정치**: 0.0678 (E9 로그 1단위 증가 → 고용률 0.068%p 증가)
- **통계적 유의성**: p = 0.281 > 0.05 (유의하지 않음)
- **95% 신뢰구간**: [-0.056, 0.191] (0을 포함하여 불확실성 높음)

#### 경제적 의미
- **E9 1% 증가 효과**: 고용률 0.0007%p 증가
- **실질적 예시**: E9 체류자 100명 → 1,000명 증가(10배) 시 고용률 0.16%p 상승
  - 즉, 고용률 70.0% → 70.16% 수준의 변화
- **정책적 해석**: 통계적으로 유의하지 않으며 경제적 효과 크기도 미미함

### 2. 제조업 집중도 효과

#### 유일한 유의한 변수
- **계수 추정치**: -7.90 (제조업 집중도 1단위 증가 → 고용률 7.9%p 감소)
- **통계적 유의성**: p = 0.027 < 0.05 (유의함)
- **해석**: 제조업 의존도가 높을수록 전체 고용률은 오히려 낮아지는 경향

### 3. 모형의 설명력

#### 낮은 설명력의 함의
- **Within R² = 0.7%**: 모형이 고용률 변동의 0.7%만 설명
- **99.3% 미설명 변동**: 고용률에 영향을 미치는 다른 요인들이 압도적으로 중요
- **모형 한계**: 현재 포함된 변수들로는 고용률 변동을 충분히 설명하기 어려움

## ✅ 결론

### 주요 발견사항

1. **E9 효과의 부재**: E9 체류자수와 지역 고용률 간에는 **통계적으로 유의한 관계가 발견되지 않음** (p = 0.281)

2. **제조업 집중도의 역설**: 제조업 집중도가 높을수록 전체 고용률은 오히려 감소하는 경향 (p = 0.027)

3. **낮은 모형 설명력**: 현재 모형으로는 고용률 변동의 0.7%만 설명 가능하여 **다른 요인들의 중요성** 부각

### 정책적 시사점

1. **신중한 정책 설계**: E9 정책은 고용률 개선의 **핵심 수단이 아닌 보조적 수단**으로 접근
2. **현실적 기대**: 과도한 정책적 기대보다는 **제한적 효과**를 전제로 한 정책 설계 필요
3. **종합적 접근**: E9 정책 단독보다는 **다양한 고용정책의 조합**을 통한 시너지 효과 모색

### 연구의 기여

본 연구는 **데이터 품질 중심의 선별적 분석**을 통해 기존 연구들에 비해 높은 내적 타당성을 확보했으나, 동시에 **정책적 효과의 한계**를 솔직하게 드러냄으로써 증거 기반 정책 설계의 중요성을 재확인했습니다.
